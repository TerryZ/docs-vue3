## 安装

将 `v-dialogs` 组件安装到项目中

::: code-group

```sh [npm]
npm i v-dialogs
```

```sh [yarn]
yarn add v-dialogs
```

```sh [pnpm]
pnpm add v-dialogs
```

:::
